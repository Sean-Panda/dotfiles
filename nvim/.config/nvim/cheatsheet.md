# Cheatsheet

## Vim Cheats

| CMD            | Description
|----------------|-----------------
| `:ggdG`        | clear file
| `:o`           | newline & insert mode  
| `:<number>`    | skip to that line
| `<shift>+v`    | select a line (add y + p to copy paste)
| `v+<up><down>` | visual mode then you can highlight multiple lines and yank them
| `dw`           | delete a word 
| `d$`           | delete all words ahead of cursor 
| `u`            | undo 
| `<ctrl>+r`     | redo 
| `<space>cd`    | open NeoVim explorer


## Plugin Cheats

#### Telescope

|     CMD     | Description          
|-------------|-----------------
| `<space>ff` | find file
| `<space>fg` | goated grep
| `<space>fb` | look at recently loaded files (might be better than harpoon)

#### Markdown

|     CMD     | Description
|-------------|-----------------
| `<ctrl>+p`  | create .md view

#### Harpoon

|     CMD     | Description
|-------------|-----------------
| `<space>+a` | add file to list
| `<shift>+e` | open harpoon menu
 
#### Treesitter
 
|     CMD     | Description
|-------------|-----------------
| `zo`        | fold under cursor 
| `zo` / `zc` | open / close fold
| `zR` / `zM` | open / close all

#### Vim Fugitive 
 
|     CMD     | Description
|-------------|-----------------
| `:G`        | git status 
| `:Gdiff`    | git diff
